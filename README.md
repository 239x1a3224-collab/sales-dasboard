📊 Sales Analytics Dashboard
📌 Project Overview

The Sales Analytics Dashboard is a modern and responsive web-based dashboard designed to visualize sales data and business performance metrics using interactive charts.
The dashboard helps users analyze sales trends, revenue growth, regional performance, product category distribution, and team productivity through a clean and user-friendly interface.

This project is built using HTML, CSS, JavaScript, and Chart.js with a modern dark-themed UI design.
Features
📈 Monthly Sales Analysis using Bar Chart
💹 Revenue Trends Visualization using Line Chart
🛍️ Product Categories Distribution using Pie Chart
🌍 Regional Sales Representation using Doughnut Chart
👥 Team Performance Comparison using Radar Chart
🌙 Modern Dark Theme Dashboard
📱 Fully Responsive Layout
⚡ Interactive and Animated Charts
🎨 Smooth Hover Effects and Card Design
🧩 Organized Dashboard Grid Layout
📊 Charts Included

Chart Type	Description

Line Chart	Revenue Trends
Pie Chart	Product Category Distribution
Doughnut Chart	Regional Sales Analysis
Radar Chart	Team Performance Comparison
